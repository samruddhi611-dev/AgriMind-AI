# 🌾 AgriMind AI

An AI-powered smart agriculture platform developed to assist farmers with crop management, disease detection, weather forecasting, and government scheme information. The application leverages Artificial Intelligence and Firebase services to provide intelligent, accessible, and data-driven farming support.

## Key Features

- AI-powered agricultural assistant
- Crop disease detection and diagnosis
- Weather-based farming insights
- AI chatbot for farmer support
- Government schemes and subsidy information
- Multilingual support
- Secure user authentication
- Responsive cross-platform application

## Installation

### Clone Repository

```bash
git clone https://github.com/samruddhi611-dev/AgriMind-AI.git
```

### Navigate to Project

```bash
cd AgriMind-AI
```

### Install Dependencies

```bash
flutter pub get
```

### Run Application

```bash
flutter run
```
